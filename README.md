# Weather Dashboard

This project is a Weather Dashboard that displays the current weather and forecast for the next four days for any city worldwide. It uses the OpenWeatherMap API to fetch weather data and displays it in a user-friendly interface. The dashboard includes a search feature, displays weather details for the current location, and current time which updates in real time.

## Features

- Search weather by city name
- Display current weather and 4-day forecast
- Show weather details for the current location
- Auto-updating time display
- Interactive temperature chart
- Background image changes based on weather conditions

## Technologies Used

- HTML
- CSS (Bootstrap)
- JavaScript (jQuery, chart.js)
- OpenWeatherMap API


## Usage

1. **Search for a city**
    - Enter the city name in the search bar and press the search icon.
    - The dashboard will display the current weather and forecast for the city.

2. **View current location weather**
    - Allow the browser to access your location.
    - The dashboard will automatically display the weather for your current location.
